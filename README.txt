Το πρόγραμμα έχει δοκιμαστεί και τρέχει κανονικά με 2 τρόπους:
	
	1) IntelliJ IDEA Compiler. Ανοίγουμε τον φάκελο src και τρέχουμε τη Main.
	Για arguments πατάμε edit configurations και τα τοποθετούμε από εκεί.

	Τρέχει και με arguments και με χωρίς
	


	2) Windows CMD και Terminal. Ανοίγουμε το path μέχρι το \src και μετα κάνουμε 	compile με javac την Main (javac Main.java),
	κάνει αυτόματα compile και τα άλλα 2 java files.
	Mετα τρέχουμε το πρόγραμμα με java Main.

	Τρέχει και με arguments και με χωρίς
	(με arguments πχ: Java Main 5 1 3 6 8 12)
